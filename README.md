# 片片公子 · 个人博客

基于纯静态技术（HTML + CSS + JS）构建的个人技术博客，兼容 GitHub Pages 部署。

## 特性

- 🌓 **暗黑/明亮双主题** — 一键切换，刷新保持
- 📊 **阅读进度条** — 实时显示页面阅读进度
- 🔍 **文章搜索 + 标签筛选** — 快速找到想要的内容
- 📑 **文章目录自动生成** — 详情页自动生成 TOC
- 💬 **评论系统** — 基于 localStorage 的本地评论
- 📱 **响应式设计** — 适配桌面端和移动端
- 🎨 **自定义光标 & 动画** — 云朵飘动、头像浮动、滚动文字

## 文件结构

```
├── index.html           # 首页
├── article-detail.html  # 文章详情页
├── style.css            # 全局样式
├── script.js            # 交互逻辑
├── avatar.jpg           # 头像
├── ShuBiao32.png        # 自定义光标
└── CNAME                # 自定义域名
```

## 使用方式

直接用浏览器打开 `index.html`，或部署到任意静态托管服务（GitHub Pages、Vercel、Netlify 等）。
